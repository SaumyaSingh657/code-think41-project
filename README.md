## Quick Start

```bash

cp frontend/.env.example frontend/.env

```

- Frontend: http://localhost:3000  
- Backend:  http://localhost:5000/api/chat  
- MongoDB:  localhost:27017

## Dev Mode

```bash
cd backend && npm install && npm run dev
cd frontend && npm install && npm run dev
```
